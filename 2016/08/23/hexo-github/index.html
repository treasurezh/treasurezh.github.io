hexo+github

仅以此博客记录在github上搭建hexo的过程。
1.安装Git和Node.js，官网即可。
在cmd中输入git –version和node -v验证两者是否安装成功。
2.打开git，输入命令npm intall -g hexo安装hexo。
3.创建一个文件夹，右击进入git bash，依次输入命令hexo init和npm install。
依次输入命令hexo generate，hexo server启动本地服务。出现
INFO Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.
的字样则说明本地服务已经完成，输入http://localhost:4000可以看到相应的hexo页面。
4.为了将hexo配置到github上，首先需要在github上新建一个repository，命名为username.github.io。（username最好不要大小写混用）
然后编辑hexo文件夹内的_config.yml文件。
deploy:
type: git
repo: https://github.com/treasurezh/treasurezh.github.io.git
branch: master
依次输入命令hexo generate和hexo deploy，出现
Branch master set up to track remote branch master from https://github.com/treasurezh/treasurezh.github.io.git.
To https://github.com/treasurezh/treasurezh.github.io.git

  • [new branch] HEAD -> master
    INFO Deploy done: git
    的字样则说明配置成功。
